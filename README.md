# Key Design Studio

Portfolio website for Key Design Studio - interior design

## Structure

```
key-design-studio/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── fonts/
└── README.md
```

## Local preview

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
